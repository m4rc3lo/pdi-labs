# Imagens sintéticas do Laboratório M1.2

Os arquivos PPM deste diretório foram criados especificamente para o
`pdi-labs`. Eles não derivam de imagens de terceiros e são distribuídos sob a
Apache License 2.0 do repositório.

- `illumination_uniform.ppm`: controle uniforme;
- `illumination_gradient.ppm`: progressão de intensidade;
- `illumination_shadow.ppm`: faixas escuras, intermediárias e claras.

O formato P3 foi escolhido por ser textual e permitir inspeção direta dos
valores RGB.
